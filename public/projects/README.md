# Project Documents Structure

This folder contains documentation for all projects (Live and WIP).

## Folder Structure

```
projects/
├── live/           # Deployed and active projects
│   ├── mycalendy/
│   ├── prismmedia/
│   ├── daosigner/
│   ├── cryptopolicy/
│   ├── notyourbot/
│   └── buildcamp/
└── wip/            # Work in progress projects
    ├── safememes/
    ├── collaborating/
    ├── cartoonmill/
    ├── indemnifi/
    ├── daostination/
    ├── planetkeeper/
    └── pollinator/
```

## Document Types

Each project folder can contain:

- **whitepaper/** - Whitepapers and research documents (.pdf, .md)
- **roadmap/** - Project roadmaps and timelines (.pdf, .md, images)
- **pitch-deck/** - Pitch decks and presentations (.pdf, .pptx)
- **technical-docs/** - Technical documentation, API docs, architecture (.md, .pdf)

## Adding Documents

1. Navigate to the appropriate project folder (live or wip)
2. Add your documents to the relevant subfolder
3. Documents will automatically appear on the website

Supported formats: `.pdf`, `.md`, `.jpg`, `.png`
